# Wwise UI (Application Control)

## Tools

- `open_project` / `close_project` — project file management
- `bring_to_foreground` / `capture_screen` — window control
- `get_selected_objects` / `get_selected_files` — read UI selection
- `execute_command` / `get_commands` — run any Wwise UI command
- `switch_layout` / `get_layout_names` / `get_current_layout_name` — layout control
- `get_view_types` / `get_view_instances` — view inspection
