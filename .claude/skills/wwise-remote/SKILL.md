# Wwise Remote (Connection Management)

## Tools

- `get_available_remote_consoles` — discover available Wwise/game instances on the network
- `connect_to_remote` — connect to a remote Sound Engine, devkit, or .prof file
- `get_remote_connection_status` — check if connected and to what
- `disconnect_from_remote` — disconnect from remote instance
